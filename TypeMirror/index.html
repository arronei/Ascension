TypeMirror

TypeMirror is our long-term bet to fix interoperability issues before they break sites by inspecting the APIs exposed by browsers to script. The IE interoperability effort uses TypeMirror to review API discrepancies between IE and Chrome/Firefox, detect new incoming issues as Chrome releases, and to prevent regressions in IE.
